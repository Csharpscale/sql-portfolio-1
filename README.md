# SQL Portfolio Project - Data Cleaning & EDA

## Overview
This repository contains two SQL projects focused on cleaning and analyzing a real-world layoffs dataset.

The goal is to demonstrate practical SQL skills used in real data analyst roles.

---
## Dataset
- **Name:** Layoffs Dataset
- **Format:** CSV
- **Source:** Kaggle
- **Description:** Contains information about company layoffs including company name, industry, country, date, layoff percentage...
- Raw data can be found in the data folder.
---

## Project 1: Data Cleaning
**File:** `sql/data_cleaning.sql`

### Tasks Performed:
- Removed duplicate records
- Standardized company and industry names
- Handled null and blank values
- cleaned date formats
- Removed unnecessary columns

This step prepares the dataset for accurate analysis.

---
## Project 2: Exploratory Data Analysis (EDA)

**File:** `sql/exploratory_data_analysis.sql`

### Analysis Includes:
- Companies with highest layoffs
- layoffs by country and industry
- Year-wise layoff trends
- Identifying patterns during economic downturns
